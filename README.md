Project Overview: “This project aims to develop a Career Recommendation System that suggests suitable career paths for students based on their academic performance.”
Functional Requirements:
•	“The system should accept and process students’ academic data, such as grades and GPA.”
•	“The system should provide career recommendations based on the analyzed academic performance.”
Testing Strategies:
	•	“Unit testing for data processing modules.”
	•	“Integration testing to ensure accurate recommendations.”
